Color correct tiff

From PanoTools.org Wiki

color_correct_tiff is an experimental utility written by Phillipe Thomin and included with hugin.

Note: this tool isn't built by default and needs to be compiled separately.

It takes two overlapping pictures and produces a third one that is version of the second where the colours match those in the first. Similar functionality is implemented in PTStitcher, PTmender and PTblender.
